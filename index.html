1

Install Claude Desktop

Download from claude.ai/download. Open it, sign in.

2

Get Kite Connect API

Visit kite.trade, sign up for Connect (₹2,000/month). Get api_key and api_secret.

3

Install Node.js

Download from nodejs.org (LTS). Verify in terminal: node --version.

4

Add MCP server

Edit ~/Library/Application Support/Claude/claude_desktop_config.json (Mac) or %APPDATA%\Claude\claude_desktop_config.json (Win) — add a zerodha-mcp entry with your API keys.

5

Daily auth

Each morning before market open, run the auth flow once. Token lasts till 7am next day.

6

Email alerts

Use the SwingEdge_Monitor.py cron script — emails you on any 3%+ drop or SL breach.
